# Future improvements

## Multi-sport expansion

Generalize the pipeline beyond football so it can support other sports (tennis, basketball, and similar) with sport-specific event models, data sources, and clip heuristics while keeping the same high-level flow (events + time alignment + cutting).

## Faster post-game availability

Reduce latency so match (or game) events and highlights can be produced soon after the final whistle—e.g. quicker ingestion of official or provider feeds, tighter caching, and workflows tuned for “straight after the game” turnaround.
